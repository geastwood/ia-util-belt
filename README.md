# IA Utility belt

## How to install

1. download the repository
`git clone git@git-1.rz1.intelliad.com:fliu/ia-util-belt.git ia`
2. make
`cd ia` // navigate to folder
`make` // make the execuable

## Dependencies
* **node** to be installed
* **ubuntu** packages
    * apt-get install inotify-tools
